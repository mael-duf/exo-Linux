# Exercice Débutant 3

## Commande utilisé 
### - `touch secret.txt`
##### Pour créer un fichier vide

### - `chmod 600 secret.txt`
##### donne les permission de lecture et écriture uniquement au propriétaire

### - `ls -l secret.txt`
##### Vérifie que les autre utilisateur ne peuvent pas le voir
